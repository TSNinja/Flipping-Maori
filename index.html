How to Use

Getting Started

Tap Get Started to load all 109 te reo Māori words and phrases. Cards are shuffled automatically each session. You can also load your own vocabulary by tapping Choose CSV File — just make sure it has two columns: Word and Definition.

Flipping Cards

Each card shows the definition face up. Tap the card to flip it and reveal the Māori word. Tap again to flip it back.

Navigating

  • ← Previous card
  • → Next card
  • Shuffle Randomise the deck and restart

When you reach the last card, pressing Next reshuffles the deck and loops back to card 1.

Keyboard Shortcuts

  • Space / Enter Flip the card
  • ← → Navigate cards

Blanked Words

If the Māori word appears within its own definition, it is replaced with ______ so you can't cheat by reading the definition!

Install as an App

To install on Android: open this file in Chrome, tap the three-dot menu and select Add to Home screen. It will run like a native app with no browser bar.
